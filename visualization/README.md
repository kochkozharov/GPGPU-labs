# Ray Tracing Scene Visualization

Проект для визуализации трехмерных сцен трассировки лучей, включая:
- Полигоны всех объектов (тетраэдр, гексаэдр, икосаэдр)
- Траекторию облета камеры
- Траекторию направления камеры

## Установка

```bash
cd visualization
uv sync
```

## Использование

### Основной скрипт

```bash
# Комбинированная визуализация (4 графика на одном изображении)
uv run python scripts/visualize.py ../cp/input/config1

# Сохранить в файл
uv run python scripts/visualize.py ../cp/input/config1 --output scene.png

# Отдельные графики (каждый в своем файле)
uv run python scripts/visualize.py ../cp/input/config1 --separate --output ./output
```

### Пример использования

```bash
uv run python example_usage.py
```

## Визуализации

Скрипт создает следующие графики:

1. **Полная сцена** - все объекты, пол, источники света и траектория камеры
2. **Траектория позиции камеры** - путь движения камеры в пространстве
3. **Траектория направления камеры** - путь точки, на которую смотрит камера
4. **Обе траектории вместе** - позиция и направление с соединительными линиями

## Зависимости
